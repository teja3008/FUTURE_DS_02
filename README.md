# FUTURE_DS_02
## 📊 Customer Churn Analysis Dashboard | Power BI

This project is an **interactive Customer Churn Analysis Dashboard** created using **Power BI** as part of my data analytics learning journey / internship work.
The dashboard focuses on analyzing customer churn patterns and identifying the key factors that influence customer retention and customer loss.

---

## 🎯 Objective

The main objective of this project is to:

- Analyze customer churn behavior
- Identify high-risk customer segments
- Understand factors affecting churn
- Track important customer retention KPIs
- Generate insights to support better business decisions

---

## 🛠️ Tools Used

- **Power BI**
- **Power Query**
- **DAX**
- **Data Modeling**
- **Data Visualization**

---

## 📂 Dataset Overview

The dataset contains customer-related information such as:

- **Customer ID**
- **Churn Status**
- **Contract Type**
- **Payment Method**
- **Tenure**
- Other customer service / subscription details

---

## 🧹 Data Preparation

The dataset was prepared and transformed using **Power Query** before building the dashboard.

### Data preparation steps included:
- Cleaning and organizing customer data
- Handling missing / inconsistent values
- Formatting columns properly
- Preparing fields for KPI and churn analysis

---

## 📈 Dashboard Features

### 🔹 KPI Cards
The dashboard includes the following key performance indicators:

- **Total Customers**
- **Churned Customers**
- **Churn Rate**
- **Average Tenure**

---

### 🔹 Visualizations Included

- **Churn Distribution**
- **Churn by Contract Type**
- **Churn by Payment Method**
- **Tenure by Contract Type**

These visuals help in understanding customer behavior and the main factors contributing to churn.

---

## 💡 Key Insights

This dashboard helps answer questions like:

- How many customers have churned?
- What is the overall churn rate?
- Which contract types are more likely to churn?
- Which payment methods are associated with higher churn?
- How does customer tenure vary across contract categories?

---
